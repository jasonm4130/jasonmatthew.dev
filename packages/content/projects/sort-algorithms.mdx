## Overview

A visual learning tool for understanding how sorting algorithms work. Select an algorithm, adjust the array size and speed, and watch it sort in real-time with color-coded comparisons and swaps.

## Features

- 25+ algorithms from bubble sort to radix sort
- Animated bar visualisations with comparison/swap highlighting
- Step-by-step pseudocode that highlights the current operation
- Sound mode that maps bar heights to audio frequencies
- Adjustable speed and array size controls

## Why I Built It

I wanted a tool that showed not just *what* sorting algorithms do, but *how* they think. Most existing visualisers show bars moving around but don't connect the visual to the logic. The pseudocode highlighting bridges that gap.
